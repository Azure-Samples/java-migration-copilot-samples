About Assets Manager

Overview

Assets Manager is a Spring Boot application that demonstrates file storage and management capabilities using AWS S3 and RabbitMQ for message processing.

Features

  • File Upload: Upload files to AWS S3 storage
  • File Management: View, download, and delete stored files
  • Message Processing: RabbitMQ integration for background processing
  • Database Integration: PostgreSQL for metadata storage

Architecture

This application consists of two main modules:

  • Web Module: Handles user interactions and file operations
  • Worker Module: Processes background tasks and message queues

Spring Boot Migration Demo

This project also serves as a demonstration for Spring Boot 2.x to 3.x migration challenges, showcasing deprecated APIs and patterns that need attention during upgrades.
