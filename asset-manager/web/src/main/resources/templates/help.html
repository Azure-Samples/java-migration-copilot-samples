Help & Instructions

How to Use Assets Manager

  1. Click on the "Upload File" button or navigate to /s3/upload
  2. Select a file from your computer (max size: 10MB)
  3. Click "Upload" to store the file in AWS S3
  4. You'll be redirected to the file list upon successful upload

Supported file types: All file types are supported, but images will display thumbnails.

  • File List: View all uploaded files at /s3
  • View File: Click on "View" to see the file content
  • Download: Click on "View" and then save the file
  • File Info: File size and upload date are displayed in the list
  1. Navigate to the file list at /s3
  2. Click the "Delete" button next to the file you want to remove
  3. Confirm the deletion (this action cannot be undone)
  4. The file will be permanently removed from AWS S3
Warning: File deletion is permanent and cannot be undone!
Common Issues:
  • Upload fails: Check file size (max 10MB) and network connection
  • AWS errors: Verify AWS credentials in application.properties
  • Database errors: Ensure PostgreSQL is running and accessible
  • RabbitMQ issues: Check RabbitMQ server status
Configuration Files:
  • application.properties - Database and AWS settings
  • logging.properties - Spring Boot 2.x logging configuration
